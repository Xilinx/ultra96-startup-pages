About

This page contains a tutorials that how user can run example project and edit the project source code

Tutorial

  • Under "Example Projects" tab user will find available example projects.

  • Go there and click on any project, it will redirect you to details page of that project.

  • For example, click on "Ultra96-V2 GPIO LEDS".

  • In details page of "Ultra96-V2 GPIO LEDS", user will find four drop down with value on,off and linux heartbeat

  • Each this dropdown will configured LED of board in any mode selected by user.

  • After that, set example run time(In seconds) from dropdown beside "Run Project" button

  • Then press "Run Project" button to run example project.

  • On board LED will be set as configured by user for specific run time.

  • User can also edit source code by clicking file available in "Source code" section.
