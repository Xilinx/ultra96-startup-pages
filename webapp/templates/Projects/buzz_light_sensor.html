{% block title %}Buzzer and Light Sensor{% endblock %}

Description

This example shows how to use the Grove light sensor and Grove buzzer. While the sensor detects light, the buzzer will remain silent. When dark, it will emit noise. For more information on the Grove Sensor Kit please visit Grover Starter Kit.

Hardware Setup

  • Attach the light sensor to A0
  • Attach the buzzer to D4
